### Lemonjsx

[![Build Status](https://travis-ci.org/dantix/lemonjsx.svg?branch=master)](https://travis-ci.org/dantix/lemonjsx)

JSX pragma for generating formatted input for lemonbar.


```jsx
<bar>
  <section
    align="l"
  >
    {left}
  </section>
  <section
    align="r"
  >
    {right}
  </section>
</bar>
```
